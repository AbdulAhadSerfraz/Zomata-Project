- Complete visualization analysis with 9 charts

- Strategic business recommendations based on data insights

## [1.0] - 2026-03-26

### Initial Release
- Set up Jupyter Notebook with exploratory data analysis
- Imported and loaded Zomato dataset
- Created 9 visualization charts:
  1. Restaurant type distribution (count)
  2. Total votes by restaurant type
  3. Rating distribution histogram
  4. Couples online order spending
  5. KDE distribution of couple spending
  6. Exact cost values analysis
  7. Order count and rating by mode
  8. Rating distribution by online availability
  9. Restaurant type vs online availability heatmap
- Generated initial README with project overview
- Set up git repository with initial commit
- Added LICENSE (MIT)
- Added .gitignore for Jupyter artifacts
